# Setup
```
clone the repo
cd into root
npm i
```
Download the [expo app](https://expo.dev/go) on your phone.

## Start the expo simulator
```
npx expo start
```
Scan the QR code above with Expo Go (Android) or the Camera app (iOS).
This should display the app.
